# Overview

This is a demonstration of my code for a tic tac toe game that I tried to make in Kotlin while learning the language in under 2 weeks.


This code is meant to run a tic tac toe game that prompts users for coordinates to place their X or O in a spot on the board.

Click the following link for a video demonstration: [Software Demo Video](https://www.youtube.com/watch?v=MBV2zQIr9s0)

# Development Environment

* Java Development Kit (JDK)
* IntelliJ
* Kotlin v1.6.10
* Git / GitHub
